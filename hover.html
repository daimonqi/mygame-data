Hover 交互系统规范 (V5)

交互基准：容器提亮 5% 白色遮罩 · 文本亮度 20% 阶梯跃升 · 品牌橙唤醒交互

1. 容器交互模式 (Container Interaction)

优先级排列：动效增强 › 基础提亮 › 纯文字。适用于卡片、列表项、磁贴容器。

动效增强 (Animated)

.hover-container-glow

交互效果：

  • 背景：无 → 5% 白色 Overlay
  • 描边：4s 匀速旋转光泽 (Mask Border)

.hover-container-base

交互效果：

  • 背景：无 → 5% 白色 Overlay
  • 无边框动效，纯净反馈

纯文字 (Text Only)

.hover-container-none

交互效果：

  • 背景：始终保持透明，不参与
  • 内部文字/图标自行产生提亮

2. 文本交互 — 白色系 (Alpha Step-up)

规则：Hover 时透明度提升一阶 (+20%)。适用于大多数可点击文本及图标。

层级 Token / 规则 静态参考 (Default → Hover) 即时演示 (请悬停)
强文本 --text-bright 100% → 持平不升 100% 保持 100% 极亮白
主正文 --text-primary 80% → 100% 80% → 100% 从 80% 升至 100%
次级文 --text-secondary 60% → 80% 60% → 80% 从 60% 升至 80%
弱化文 --text-tertiary 40% → 60% 40% → 60% 从 40% 升至 60%
幽灵态 --text-ghost 20% → 40% 20% → 40% 从 20% 升至 40%
蓝色链接 #409EFF → #66B1FF + 下划线 → 外部跳转链接演示

3. 文本交互 — 品牌系 (Brand Colorize)

规则：Hover 时激活品牌橙 #FF7210。适用于导航菜单、核心功能入口、主令按钮文字。

类型 实现类名 默认状态 即时演示 (请悬停)
品牌文本 .text-brand 品牌橙 #FF7210 悬停变浅橙 #FF933B
唤醒态 .hover-awaken 80% 白色文本 悬停唤醒品牌橙 AWAKEN
