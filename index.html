PC 233 Design System

Button System (PC-233)

定义了从基础业务按钮到高保定启动按钮的完整状态机。

1. 高保定启动按钮 (High-Fidelity)

适用于游戏启动、核心下载、充值等顶级动作。具有环境光弥散与内部描边。

Class: .btn-hifi
视觉要素：
- 渐变：双层径向渐变叠加 (Radial Gradient)
- 光效：Inset Shadow 模拟内发光 + 外部弥散投影
- 描边：1px 半透明线性渐变 Mask 描边
- 交互：Hover 提亮并略微放大，Active 缩放回弹

2. 基础业务按钮 (Standard)

Primary Button

主操作，品牌色背景。

.btn-primary

Disabled State

禁用态，40% 透明度。

Hover Interaction System (V4)

交互基准：容器提亮 5% 白色遮罩 | 文本亮度 20% 级阶跃升 | 品牌橙唤醒交互

1. 容器交互模式 (Containers)

动效增强 (Animated)

.hover-container-glow

容器提亮 + 旋转流光边框

基础提亮 (Basic)

.hover-container-base

容器 5% 白色遮罩提亮

纯文字模式 (Text Only)

.hover-container-none

仅内部文字/图标产生交互

2. 文本交互刻度 (Typography Matrix)

类型 变量 跃迁规则 实时交互 (Hover Me)
主正文 --text-primary 80% → 100% 悬停提亮至 100%
次级文 --text-secondary 60% → 80% 悬停提亮至 80%
蓝色链接 --color-link-blue #409EFF → #66B1FF 交互式超链接
唤醒态 .hover-awaken White → Brand Orange 唤醒品牌颜色
